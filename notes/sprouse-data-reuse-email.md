# Draft Email: Sprouse Data Reuse

To: Jon Sprouse <jon.sprouse@nyu.edu>

Subject: Question about reusing 2013/2017 acceptability judgment data

Dear Professor Sprouse,

I am working on a secondary-data reassessment of magnitude estimation in
acceptability judgment research, centered on Bard, Robertson, and Sorace (1996)
and later method-comparison studies.

I found the public data files for Sprouse, Schuetze, and Almeida (2013) and
Sprouse and Almeida (2017) on your website. The file headers say that the data
are provided so researchers can verify the analyses, and ask researchers to
contact you about possible use for novel research.

Would you be comfortable with my using these public files for a new secondary
analysis comparing magnitude estimation with Likert, forced-choice, and yes/no
judgment formats? The planned use is limited to aggregate reanalysis and
methodological comparison. I would not redistribute the raw participant-level
files, and I would cite the original papers and data source clearly.

The current framing is deliberately conservative: the analysis would treat the
methods as noisy observations of a common item-level acceptability signal and
ask whether magnitude estimation shows any practical method-specific advantage
after accounting for item and task structure.

Please let me know whether this use is acceptable, and whether there are any
conditions, preferred citations, or caveats you would want me to observe.

Best,

Brett Reynolds
